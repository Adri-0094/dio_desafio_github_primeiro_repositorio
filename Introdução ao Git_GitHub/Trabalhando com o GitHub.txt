git config --list (lista as configurações salvas)

git config --global --unset user.email - email (exemplo de modificação)
(para resetar e alterar configuração)

Ao criar o repositório no GitHub, no Bash do Git utilizamos o comando:
git remote add origin https://github.com/Adri-0094/dio_desafio_github_primeiro_repositorio
(exemplo da URL desse diretório criado)(origin é um alias)
para inclusão desse repositório local para o remoto.

git remote -v (lista os repositórios remotos que temos)

git push origin master (empurra os arquivos do repositório local para o remoto)
